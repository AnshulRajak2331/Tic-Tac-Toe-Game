# Tic-Tac-Toe-Game
This is a simple Tic-Tac-Toe game built using Python and Tkinter. It features a clean graphical interface with two-player gameplay. The game automatically detects and highlights the winning combination.


## Features
- Two-player gameplay (X and O)
- Interactive Tkinter GUI
- Automatic winner detection
- Winning cells highlighted

## Technologies Used
- Python
- Tkinter

## Run the Project

```bash
python main.py
```
